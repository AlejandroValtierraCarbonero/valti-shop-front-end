# valti-shop-front-end
Proyecto de práctica. El Front-end de una tienda

## Stack
* El proyecto está desarrollado en VUE 3 con Vuetify para los elementos visuales
* TypeScript
* Vue router y pinia
* VUE 3 composition API

## Conventions
* todo el proyecto está en inglés
* La estructura de carpetas estará oriendata a describir el funcional más que los compoenntes técnicos reutilizables

## Arquitectura
El proyecto coniste en una SPA que conecta con una API Rest
