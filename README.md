# valti-shop-front-end

Proyecto de práctica. El Front-end de una tienda

## Stack

* El proyecto está desarrollado en VUE 3 con Vuetify para los elementos visuales
* TypeScript
* Vue router y pinia
* VUE 3 composition API

## Conventions

* todo el proyecto está en inglés
* La estructura de carpetas estará oriendata a describir el funcional más que los compoenntes técnicos reutilizables

## Arquitectura

* El proyecto coniste en una SPA que conecta con una API Rest
* El proyecto será enteramente responsive y tendrá versión tanto para ordenadores, como tables como móviles

## Repositorio

* Se seguirán las buenas prácticas de gestión de repositoio con la metodología gitflow

## Iniciar el proyecto

Para inciar el proyecto en local hay que ejecutar los siguientes comandos

* cd vue-project
* npm i
* npm run dev

## compilar el proyecto

Para compilar el proyecto y crear los artefactos para su publicación hay que ejecutar los siguientes comandos

* cd vue-project
* npm i
* npm run build
